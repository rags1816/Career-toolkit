# Career Toolkit

A set of tools supporting structured, evidence-based career development and
job application work — built from methodology developed while producing
senior procurement/product management applications (Home Office, Ministry of
Justice, Monzo, and general procurement director roles).

## What it does

- Helps structure job applications around evidence-based competency criteria
  (e.g. PBMG, PROD, INOV-style frameworks) rather than generic claims
- Draws on a structured career evidence bank (e.g. Castrol India supply chain
  transformation, BP category portfolios, NESO digital procurement) to match
  the right evidence to the right criterion
- [Add as developed — e.g. CV tailoring assistant, cover letter structuring,
  interview STAR-answer builder, gap-honesty checker]
